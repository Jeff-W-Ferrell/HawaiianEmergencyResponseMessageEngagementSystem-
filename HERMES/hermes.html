H.E.R.M.E.S.

Hawaii Emergency Response Message Engagement System

Test Description

Initiating TEST will utilize the H.E.R.M.E.S. system to send the following text message to all Hawaiian residents within cell phone range: "THIS IS A TEST OF THE HAWAIIAN EMERGENCY RESPONSE MESSAGE ENGAGEMENT SYSTEM. THIS IS ONLY A TEST."

ACTUAL Description

Initiating an ACTUAL MISSILE THREAT ALERT will utilize the H.E.R.M.E.S. system to send the following text message to all Hawaiian residents within cell range: "INCOMING BALLISTIC MISSILES HAVE BEEN DETECTED AND VERIFIED. THIS IS NOT A TEST. TAKE COVER IMMEDIATELY."
